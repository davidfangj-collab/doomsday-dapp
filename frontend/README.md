# Doomsday Message Board - 前端使用指南

## 🚀 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 配置合约地址

在 `src/App.jsx` 中，将 `CONTRACT_ADDRESS` 替换为你部署的合约地址：

```javascript
const CONTRACT_ADDRESS = "0x..." // 你的合约地址
```

### 3. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

## 📦 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录。

## 🎯 使用说明

### 连接钱包
1. 点击右上角"连接钱包"按钮
2. 在 MetaMask 中确认连接
3. 连接成功后显示钱包地址

### 发布留言
1. 在输入框中输入留言内容（最多280字符）
2. 设置竞价金额（必须大于最低竞价）
3. 点击"发布留言"按钮
4. 在 MetaMask 中确认交易
5. 等待交易确认

### 查看排行榜
- 自动显示 Top 100 留言
- 前三名有特殊徽章（金、银、铜）
- 显示竞价金额和发布时间
- 悬停卡片有动画效果

## 🔧 自定义配置

### 修改配色

编辑 `src/index.css` 中的 CSS 变量：

```css
:root {
  --accent-primary: #ff0055;    /* 主色 */
  --accent-secondary: #00ffff;  /* 辅色 */
  --accent-tertiary: #ffaa00;   /* 强调色 */
}
```

### 修改动画速度

在 `src/App.jsx` 中调整 Framer Motion 的 `transition` 参数：

```javascript
transition={{ duration: 0.8, delay: 0.2 }}
```

### 修改粒子数量

在 `createParticles` 函数中修改循环次数：

```javascript
for (let i = 0; i < 50; i++) { // 修改这个数字
  // ...
}
```

## 🐛 常见问题

### MetaMask 未安装
- 提示：请安装 MetaMask 钱包
- 解决：访问 https://metamask.io 安装

### 交易失败
- 检查钱包余额是否充足
- 检查竞价是否大于最低竞价
- 检查网络是否正确（Sepolia 测试网）

### 合约地址未配置
- 确保已部署合约
- 在 `App.jsx` 中配置正确的合约地址
